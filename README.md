# Metodologías-2026

Repositorio destinado a la entrega y organización de los trabajos prácticos desarrollados durante la cursada de Metodologías de Programación I (2026).

## Objetivo
En este repositorio se subirán los trabajos prácticos dados en clase, junto con ejercicios, resoluciones y material relacionado con la materia.

## Contenido
Cada trabajo práctico estará organizado en carpetas, por ejemplo:

- TP1 - Introducción y Tipos Abstractos de Datos
- TP2 - Interfaces y Programación Orientada a Objetos
- TP3 - Recursión y Excepciones
- TP4 - Estructuras de Datos
- TP5 - Corrección y Eficiencia de Algoritmos

## Tecnologías utilizadas
- C#
- SharpDevelop
- .NET Framework 4.8
